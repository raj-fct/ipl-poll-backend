# IPL Poll — Flutter App

A Flutter mobile app for predicting IPL match outcomes and winning coins.

## Quick Start
See **SETUP.md** for full instructions.

1. Update `lib/core/constants.dart` with your API URL
2. Run `flutter pub get`
3. Run `flutter run`

## Default Login (after backend is set up)
- **Mobile**: `9999999999`
- **Password**: `Admin@123` (admin account)
